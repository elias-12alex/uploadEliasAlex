Domótica:

Primeiramente, para entender sobre as ferramentas necessárias para realizar um sistema de domótica, é fundamental compreender o conceito de automação residencial, que é a capacidade de criar, programar e agendar eventos para o funcionamento automático de equipamentos em uma casa. Utilizando sistemas inteligentes ou integrados em uma controladora de automação, é possível monitorar e gerenciar toda a residência através de uma central via smartphone, tablet e internet. Para a criação de um sistema inteligente é conveniente estudar sobre ferramentas, como arduíno e raspberry pi, que são instrumentos de baixo custo e acessíveis tanto para profissionais em programação e eletrônica quanto para curiosos que gostam do tema. Mas afinal, o que é um arduíno e um raspberry pi? E quais suas funcionalidades?

Bom, segundo o cientista da computação Adilson Thomsen, arduino (figura 2) é uma a placa composta por um microcontrolador, circuitos de entrada/saída e que pode ser facilmente conectada à um computador e programada via IDE (Integrated Development Environment, ou Ambiente de Desenvolvimento Integrado) utilizando uma linguagem baseada em C/C++. Já o raspberry pi (figura 3) é um minicomputador, semelhante ao PC (personal computer). A diferença é que este dispositivo é compacto e possui todos os principais componentes de um computador numa pequena placa do tamanho de um cartão de crédito. Ele, assim como o arduino, pode ser utilizado para automatização de tarefas através de programação via lDE. Porém, antes de continuar o assunto sobre como essa programação é feita, necessita-se saber mais sobre o que é programação e quais as linguagens, geralmente, usadas para estes fins.

O que é programação?

A programação ligada ao mundo tecnológico tem uma definição bastante complexa. Segundo o site idocode, a programação é um processo de escrita, testes e manutenção de programas de computadores. Esses programas, por sua vez, são compostos por conjuntos de instruções determinados pelo programador que descrevem tarefas a serem realizadas pela máquina e atendem diversas finalidades. As instruções do programador para a máquina são feitas a partir de linguagens de programação.

Quais são as linguagens de programação utilizadas? E suas aplicações no sistema?

Existem centenas de variações e todas elas permitem que o programador defina sobre quais dados o computador deverá atuar, como armazenar e transmitir esses dados e quais ações tomar sob variadas circunstâncias. As principais linguagens de programação aplicadas para fazer esta comunicação são as linguagens C/C++ e PHP. As linguagens C/C++ servem para manipular as ações na IDE do arduino, a PHP serve para fazer a conexão via servidor web.
